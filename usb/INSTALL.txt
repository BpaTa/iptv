   УСТАНОВКА IPTV НА ТЕЛЕВИЗОР  (читать с самого телевизора)
═══════════════════════════════════════════════════════════════

Что у тебя на флешке в папке IPTV/:
  • playlist.m3u      — плейлист с каналами
  • StreamVault.apk   — приложение-плеер  (может отсутствовать,
                        тогда скачай отдельно — см. ниже)
  • INSTALL.txt       — этот файл

───────────────────────────────────────────────────────────────
ШАГ 1. Установить StreamVault
───────────────────────────────────────────────────────────────

Открой на ТВ файловый менеджер (X-plore, File Commander и т.п.
из Google Play, если предустановленного нет).

Перейди:  USB → IPTV → StreamVault.apk → нажми

ТВ скажет "установка из неизвестных источников запрещена" —
зайди в настройки по подсказке, разреши для файлового менеджера,
вернись и установи.

Если APK не было на флешке, скачай его на ТВ через приложение
"Downloader" (поставь из Play Store):
   github.com/Davidona/StreamVault-IPTV/releases/latest

───────────────────────────────────────────────────────────────
ШАГ 2. Скопировать playlist.m3u во внутреннюю память ТВ
───────────────────────────────────────────────────────────────

Это нужно, чтобы плейлист продолжал работать даже когда флешка
вынута.

В файловом менеджере:
   USB → IPTV → playlist.m3u → копировать
   Внутренняя память → Download → вставить

───────────────────────────────────────────────────────────────
ШАГ 3. Подключить плейлист в StreamVault
───────────────────────────────────────────────────────────────

Открой StreamVault. При первом запуске будет мастер настройки:

   Source type:      M3U
   Source location:  Local file
   File path:        /storage/emulated/0/Download/playlist.m3u
                     (или через "Browse" найди файл руками)

Жми Add / Sync. Через 5-10 секунд появятся 9 категорий и
~38 каналов.

───────────────────────────────────────────────────────────────
ШАГ 4. (необязательно) Удобство
───────────────────────────────────────────────────────────────

  • Long-press на категории → Pin / Hide
  • Long-press на канале    → Add to Favorites
  • Settings → Parental Controls — выставь PIN

───────────────────────────────────────────────────────────────

EPG (программа передач) подтянется автоматически из плейлиста.
Если не подтянулся:
   Settings → EPG Sources → Add Source
   URL: https://iptvx.one/EPG.xml.gz
