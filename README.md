# Русский IPTV для ТВ — самообновляемый плейлист

Большая курированная сборка из **~193 каналов в 18 категориях** для просмотра через **StreamVault** на Android TV. Полностью автоматическая: один раз настроил — и забыл, плейлист обновляется ежедневно на серверах GitHub без твоего участия.

Без эротики, без 18+, без шопинга, без шлака — только то, что обычно смотрят. С премиальными мировыми брендами: **Discovery / National Geographic / History / BBC Earth / Disney / Nickelodeon / Fox Sports / Premier Sports / Stingray / Paramount**.

## Установка — две команды

### Вариант 1: «Полная автоматика» (рекомендуется) — IPTV на годы без ухода

```bash
./setup-github.sh
```

Что делает:
1. Создаёт публичную GitHub-репу с этим кодом
2. Активирует GitHub Action, который пересобирает плейлист **каждый день в 06:17 МСК**
3. Печатает тебе URL — что-то вроде `https://raw.githubusercontent.com/ТЫ/iptv/main/playlist.m3u`

Дальше на ТВ один раз:
1. Поставь **StreamVault** (см. ниже)
2. Add Source → **M3U URL** → вставь URL
3. Поставь галку **Auto-refresh: daily**
4. Готово. **Никогда больше ничего трогать не надо.** Когда iptv-org обновит ссылки — твой workflow это подхватит, StreamVault на ТВ скачает обновлённый плейлист на следующий день.

### Вариант 2: Через флешку (если без GitHub)

```bash
./update.sh /Volumes/ИМЯ_ФЛЕШКИ
```
Пересоберёт плейлист → скопирует на флешку → переносишь на ТВ → ставишь как локальный файл в StreamVault. Минус — раз в месяц нужно повторять.

## Установка StreamVault на ТВ (одноразово)

1. Скачай APK на компьютер:
   `https://github.com/Davidona/StreamVault-IPTV/releases/latest`
2. Положи `StreamVault.apk` рядом с этим README — он попадёт на флешку через `update.sh` автоматически.
3. На ТВ открой файловый менеджер (X-plore / File Commander) → флешка → `IPTV/StreamVault.apk` → установить.
4. ТВ запросит «разрешить установку из неизвестных источников» — разреши для файлового менеджера.

## Источники данных

- **[iptv-org/iptv](https://github.com/iptv-org/iptv)** — главный открытый каталог IPTV (60k+ звёзд, обновляется сообществом несколько раз в неделю). Используем `countries/ru.m3u` для русских каналов и `categories/religious.m3u` для исламского блока.
- **EPG**: `iptvx.one/EPG` — открытая программа передач с фокусом на русские каналы.

## Логика отбора

1. **Белый список** в `channels.py` — каждый канал описан regex-патерном, по которому ищем в источнике, и предпочтением по качеству (1080 > 720 > 576 > 540 > 450). Если канал есть в нескольких вариантах — выберется самый качественный.
2. **Чёрный список** (`BLOCKLIST_PATTERNS`) — даже если что-то взрослое попадёт в источник, отсечётся по подстроке (18+, эротика, playboy, и т.д.).
3. **Geo-restrictions:** примерно треть российских каналов гео-блокируются вне РФ (СТС, РЕН, ТВ-3, Муз ТВ, Спас и т.п.). За пределами России могут не играть. Внутри РФ — работают все.
4. **Исламский блок намеренно компактный:** только английский (Peace TV English, Huda TV) и арабские трансляции главных мечетей (Al Quran Al Kareem из Мекки, Al Sunnah из Медины). Без urdu/bangla/китайского — для пользователей, читающих по-русски и английски.

## Как обновлять / добавлять каналы

### Добавить канал

1. Открой `https://iptv-org.github.io/iptv/countries/ru.m3u` в браузере → Cmd+F → найди точное имя канала.
2. В `channels.py` добавь элемент в нужную категорию:
   ```python
   {"name": "Мой канал", "patterns": [r"^Точное имя из источника"], "prefer": "1080"},
   ```
3. `python3 build.py` → проверь, что собрался без ошибок → `git push`. GitHub Action подхватит.

### Канал в источнике переименовался

Открой `channels.py`, поправь регулярку. То же самое.

### Когда часть каналов «отвалится»

IPTV — это бесплатные публичные потоки, провайдеры периодически меняют URL. Раз в 2-4 недели 1-2 канала могут перестать играть. С автоматикой через GitHub — ничего делать не нужно: на следующий день iptv-org обновит ссылку, твой workflow пересоберёт плейлист, StreamVault скачает новый.

## Зависимости

- macOS / Linux
- Python 3.8+ (есть на macOS из коробки)
- `curl`, `git` (есть в macOS из коробки)
- `gh` (только для `setup-github.sh`) — `brew install gh && gh auth login`
- **Никаких pip-пакетов.** Скрипт использует только стандартную библиотеку.

## Хочешь убрать категорию или канал

Открой `channels.py`, закомментируй или удали запись → `python3 build.py` → `git commit -am "..." && git push`. Всё.
